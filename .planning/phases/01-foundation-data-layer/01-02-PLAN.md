   ```bash
   npx shadcn@latest init --defaults
   ```
   This creates:
   - `components.json` configuration
   - `src/lib/utils.ts` with cn() helper
   - Updates to `src/app/globals.css` with CSS variables

   Use --defaults flag to accept: New York style, Zinc base color, CSS variables enabled.

2. Add the Button component:
   ```bash
   npx shadcn@latest add button --yes
   ```
   This creates `src/components/ui/button.tsx`

3. Verify `src/lib/utils.ts` contains the cn() function:
   ```typescript
   import { clsx, type ClassValue } from "clsx"
   import { twMerge } from "tailwind-merge"

   export function cn(...inputs: ClassValue[]) {
     return twMerge(clsx(inputs))
   }
   ```

Note: The --defaults and --yes flags ensure non-interactive installation.

1. `components.json` exists with shadcn configuration
2. `src/components/ui/button.tsx` exists
